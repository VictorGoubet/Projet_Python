
Projet python ESILV 2020  - METHODE D'INSTALLATION

Le dossier livrable est un dossier regroupant tout l'essentiel du projet, vous etes donc invité à consulté ce dossier

<------PREREQUIS------->

	Afin que cette WebApp fonctionne correctement veuillez prendre connaissance du fichier "requirements.txt" 
	listant tout les modules necessaires au bon fonctionnement de WicoronApp
	Tout les modules sont installables avec pip 


<------LANCEMENT------->

  	-Lancer anaconda prompt ou un autre shell et déplacez vous dans le dossier "WebApp" (cd ..)
	-Si vous utiliser un environnement activez le
  	-Une fois au bon endroit taper la commande "python manage.py runserver"
  	-Aller sur votre navigateur et taper dans la barre url "localhost:8000"
  	-Bienvenue sur WicoronApp !
