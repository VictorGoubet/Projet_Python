Projet python ESILV 2020

Pour faire contionner la WebApp:

  -Telecharger Django si vous ne l'avez pas (conda install django dans anaconda prompt )
  -Lancer anaconda prompt ou un autre shell et vous déplacer dans le dossier "WebApp" 
  -(Pour se déplacer dans un dossier depuis la console : cd <chemin vers le dossier> )
  -Une fois au bonne endroit taper la commande "python manage.py runserver"
  -Aller sur internet et taper dans la barre url "localhost:8000"
  -Vous êtes sur le site


Packages necessaires:

-Folium (visualisation)
-requests
-Django
